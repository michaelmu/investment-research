This is a lightweight checklist we apply to candidates to force discipline on **entry valuation** and **incremental economics**.

It’s inspired by a multibagger-factor writeup you shared (Yartseva 2009–2024), but we treat it as a **lens**, not a rigid screen.

**Not financial advice.**

## Multibagger lens (quick checklist)

### 1) Valuation + cash generation
- **FCF positive?** (and not just one year)
- **FCF yield** (FCF / market cap) — if we can estimate it from filings + price/shares.
  - If **≥ ~5%**, call it out as an “interesting entry point” signal.
  - If **< ~2–3%**, you’re relying on perfection or very long-duration growth.

### 2) Balance sheet sanity
- **Negative equity?** Treat as a major red flag unless we can explain it clearly.

### 3) Improving economics
- Are **gross / operating / FCF margins** improving over 3–5 years?
- Are returns (ROIC/ROCE proxies) improving or at least stable?

### 4) Reinvestment quality (proxy)
- Does **asset growth** appear to be running **ahead** of earnings power growth (EBITDA/operating income)?
- If assets are ballooning without earnings improvement, treat as a “bad reinvestment” smell.

### 5) Price context (context, not a signal)
- Where is the stock relative to its **12‑month range**?
- Note any “dull moment” entry point, but don’t overfit momentum rules.

## How we use this
Every deep dive includes a short **Multibagger lens** section:
- FCF yield (if estimable)
- equity sanity check
- improving economics notes
- asset growth vs earnings power notes
- 12‑month range position

If we can’t estimate a metric quickly, we label it **unknown** and move on.
